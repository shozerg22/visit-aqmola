# Аналитика и скоринг

## Сбор данных (DataHub)
- Посещаемость объектов (просмотры/визиты)
- Загрузка маршрутов и активности
- Жалобы и отзывы
- Бронирования (обезличенная статистика)
- Прогноз загруженности (модель позже)

## Эндпоинты (черновик)
- `POST /api/v1/metrics/ingest` — приём метрик из фронтенда/сервисов

## Скоринг объектов (MVP)
- Метрики: средний рейтинг и число жалоб
- Формула (простая):
  - `base = (avg_rating / 5) * 100`
  - `penalty = min(complaints * 3, 30)`
  - `score = max(0, base - penalty)`
- Эндпоинт: `GET /api/v1/objects/{id}/score`
