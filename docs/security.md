# Безопасность и соответствие

## Аутентификация / Авторизация
- OIDC через eGov Mobile (OAuth2)
- Сессии/кэш — Redis
- Роли: user, business, admin

## Защита API
- WAF и rate limiting (на уровне шлюза)
- HTTPS везде, HSTS
- Логи и аудит админ-операций

## Персональные данные
- Минимизация: хранить только необходимое
- Шифрование в покое: уровне СХД/KMS
- Журналы доступа и инцидентов

## Секреты и конфигурация
- ENV-переменные, секрет-менеджер
- Раздельные ключи для sandbox/prod

## Наблюдаемость
- Метрики/трейсинг, алерты (Prometheus/Grafana)
- Логи приложений и шлюза
